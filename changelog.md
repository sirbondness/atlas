# Changelog

All notable changes to this project will be documented in this file.

The format is based on **Keep a Changelog** and this project adheres to **Semantic Versioning**.

---

## [0.2.0] - 2026-06-30

### Added

* Docker Inventory API
* Docker Metrics API
* Docker search functionality
* Docker filtering
* Docker sorting
* Interactive Docker container detail drawer
* Reusable Card component system
* Atlas API client abstraction
* Container query hook
* GitHub Project board
* Development roadmap
* Project documentation

### Changed

* Refactored backend into a service-based architecture
* Split Docker Inventory and Metrics into dedicated services
* Introduced reusable frontend hooks
* Improved Docker dashboard user interface
* Improved component structure and code organization

### Fixed

* Reduced Docker API response latency
* Eliminated frontend bottlenecks caused by expensive Docker operations
* Improved project maintainability through architectural refactoring
